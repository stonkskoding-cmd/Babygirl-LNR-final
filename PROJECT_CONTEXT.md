# 🎯 ПРОЕКТ: BABYGIRL_LNR

## 📌 ОПИСАНИЕ
Сайт эскорт-агентства для ЛНР (города: Луганск, Стаханов, Первомайск)

## 🏗️ СТЕК
- **Backend**: Node.js + Express
- **Database**: MongoDB Atlas
- **Frontend**: Vanilla JS (HTML/CSS/JS в одном файле public/index.html)
- **Deploy**: Render.com
- **Auth**: JWT + bcryptjs

## 🔑 ФУНКЦИОНАЛ

### 1. КАТАЛОГ ДЕВУШЕК
- Сетка карточек с фильтрами по городам
- Модалка профиля с галереей фото
- CRUD в админке

### 2. ЧАТ С БОТОМ (FSM)
**Состояния:**
- greet → asking_city → picking_girl → picking_service → waiting_operator

**Сценарий:**
1. Бот здоровается, просит город
2. Клиент пишет город → бот показывает кнопки с девушками
3. Клиент выбирает → бот показывает анкету + кнопки услуг
4. Клиент выбирает услугу → заявка оператору
5. Оператор отвечает → клиент видит

**ВАЖНО:**
- Регистронезависимый поиск (луганск/ЛУГАНСК/Луганск)
- Авто-обновление каждые 3 секунды
- История сохраняется

### 3. АДМИН-ПАНЕЛЬ
- **Вкладка "Анкеты"**: CRUD девушек, загрузка фото
- **Вкладка "Чаты"**: список чатов, ответы оператора
- **Вкладка "Настройки"**: заголовки, телефон

### 4. АВТОРИЗАЦИЯ
- Роли: admin, client
- JWT токены
- Два оператора:
  - admin_main / Babygirl2024!
  - operator_01 / Operator2024!
